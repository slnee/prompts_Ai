## Task
The aim of this project was to do a web page to show a prompt AI library and to learn to do a commit chain

## Description
I create a web page which shows the content of a json file, which is some prompt for ai.
There is a form we can complete to add another prompt but this one will disappear when we reload the page.

## Installation
For the project to work, we need to install docker.
in VScode, do a git clone to recover the repository.
To start the server, write "docker compose up" in the terminal.
And to access the page, in a browser, write "localhost:3000".
